## Network emulator 
[**Mininet**](https://mininet.org/)

[**Open vSwitch**](https://www.openvswitch.org/)

## Container
[**Docker**](https://www.docker.com/)
### Version:
1. Docker:
    - client: `28.4.0`
    - server: `28.4.0`
2. Docker compose: `v2.39.2`


## Steps to setup
1. make sure on the right location:
```bash
cd onos-testbed
```
2. start the docker compose:
```bash
docker compose up
```
